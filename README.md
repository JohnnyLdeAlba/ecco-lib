# EccoLib

A graphics extraction library for the Ecco the Dolphin series.

# Supported Games
- Ecco the Dolphin
-- Sprites, Stage Sprites and Level Maps
- Ecco the Dolphin PC
-- Sprites and Level Maps
- Ecco 2: The Tides of Time
-- Sprites, Stage Sprites and Level Maps
- Ecco Jr 
-- Sprites and Level Maps
